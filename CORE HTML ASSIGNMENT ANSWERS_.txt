        <dt>Frontend:</dt>
        <dd>
            <p>Frontend technologies are responsible for the user interface and user experience of a web application.</p>
            <ul>
                <li>HTML (HyperText Markup Language): Markup language for creating the structure of web pages.</li>
                <li>CSS (Cascading Style Sheets): Used for styling and layout of web pages.</li>
                <li>JavaScript: The programming language for adding interactivity and dynamic behavior to websites.</li>
                <li>React: A JavaScript library for building user interfaces.</li>
                <li>Vue.js: A progressive JavaScript framework for building user interfaces.</li>
            </ul>
        </dd>


        <dt>Backend:</dt>
        <dd>
            <p>Backend technologies handle server-side logic, databases, and server operations.</p>
            <ul>
                <li>Node.js: A runtime environment for executing JavaScript on the server.</li>
                <li>Express.js: A web application framework for Node.js.</li>
                <li>Python: A versatile programming language often used for backend development.</li>
                <li>Django: A high-level Python web framework.</li>
                <li>Ruby: Another programming language often used for web development.</li>
                <li>Ruby on Rails: A web application framework for Ruby.</li>
            </ul>
        </dd>


        <dt>Database:</dt>
        <dd>
            <p>Databases store and manage data for web applications.</p>
            <ul>
                <li>SQL (Structured Query Language): A language for managing relational databases.</li>
                <li>MySQL: An open-source relational database management system.</li>
                <li>MongoDB: A NoSQL database for handling unstructured or semi-structured data.</li>
                <li>PostgreSQL: An open-source relational database system known for its advanced features.</li>
            </ul>
        </dd>


        <dt>Server:</dt>
        <dd>
            <p>Servers host web applications and serve them to users.</p>
            <ul>
                <li>Nginx: A popular web server known for its high performance and scalability.</li>
                <li>Apache: A widely-used open-source web server.</li>
            </ul>
        </dd>


        <dt>Version Control:</dt>
        <dd>
            <p>Version control systems help manage and track changes to code.</p>
            <ul>
                <li>Git: A distributed version control system.</li>
                <li>GitHub: A web-based platform for hosting and collaborating on Git repositories.</li>
                <li>Bitbucket: A web-based platform for Git and Mercurial version control.</li>
            </ul>
        </dd>
    </dl>
</body>
